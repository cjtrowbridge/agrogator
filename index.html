Agrogator

Show/Hide Story Contents

This free script helps aggregate rss feeds in-browser using Google's free Feeds API and JQuery.

To change, add or remove feeds, simply add amnother object with class "agrogator_enclosure" and a data-uri pointing to the rss feed you want to add.

All feeds are fetched, up to the maximum number of entries set in the included script, and then added to the container and sorted by pubdate.

Enjoy!
